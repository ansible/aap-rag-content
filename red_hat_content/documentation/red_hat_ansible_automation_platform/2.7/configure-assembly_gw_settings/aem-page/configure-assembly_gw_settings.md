# Configure Ansible Automation Platform

You can configure Ansible Automation Platform from the **Settings** menu by using the following selections:

-  **Subscriptions**
-  **Platform gateway**
-  **User Preferences**
-  **Troubleshooting**


Note:

The other selections available from the **Settings** menu are specific to automation execution.
