# Discover and manage execution environments and collections

Ansible automation portal provides catalog views for browsing execution environment definitions and Ansible collections discovered from configured content sources.

The execution environment catalog shows all execution environment definitions registered in Ansible automation portal. Use the catalog to browse definitions, view their configurations, trigger builds, and download definition files.

The collection catalog shows Ansible collections discovered from configured content sources, including Git repositories and private automation hub. Browse collections to view metadata, readmes, dependencies, and links to source repositories.

## Execution environment catalog actions

From the execution environment catalog, you can perform the following actions:

- Browse and search for execution environment definitions by name.
- Filter definitions by owner and tag.
- Star definitions for quick access through the **Starred** filter.
- View definition details including metadata, defined content, readme, and resources.
- Trigger builds on definitions saved to a GitHub repository.
- Download definition files as a `.tar` archive.
- Delete or unregister definitions from the catalog.
