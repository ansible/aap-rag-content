# Scale with automation mesh in a containerized or VM environment

Run automation on different execution nodes across your data center, clouds, or remote sites with automation mesh.

For RPM or Containerized VM-based Ansible Automation Platform you use an installation program to set up automation mesh or to upgrade automation mesh. To provide Ansible Automation Platform with details about the nodes, groups, and peer relationships in your mesh network, you define them in an the inventory file in the installer bundle.
