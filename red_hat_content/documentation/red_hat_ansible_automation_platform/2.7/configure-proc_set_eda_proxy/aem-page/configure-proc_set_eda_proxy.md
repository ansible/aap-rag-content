# Configure proxy settings for Event-Driven Ansible

For Event-Driven Ansible, there are no global settings to set a proxy. You must specify the proxy for every project.

## Procedure

1.  From the navigation panel, select Automation Decisions> (and then)Projects.
2.  Click Create Project
3.  Use the **Proxy** field.
