# Automation controller variables

Inventory file variables for automation controller.

| Variable name                               | Description                                                                                                                                                                                                                                   | Required or optional                                         | Default                                                                        |
| ------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------------------------ |
| <br>`controller_admin_email`                | <br>Email address used by Django for the admin user for automation controller.                                                                                                                                                                | <br>Optional                                                 | <br>`admin@example.com`                                                        |
| <br>`controller_admin_password`             | <br>Automation controller administrator password. Use of special characters for this variable is limited. The password can include any printable ASCII character except `/`, `"`, or `@`.                                                     | <br>Required                                                 |                                                                                |
| <br>`controller_admin_user`                 | <br>Username used to identify and create the administrator user in automation controller.                                                                                                                                                     | <br>Optional                                                 | <br>`admin`                                                                    |
| <br>`controller_create_preload_data`        | <br>Controls whether or not to create preloaded content during installation.                                                                                                                                                                  | <br>Optional                                                 | <br>`true`                                                                     |
| <br>`controller_event_workers`              | <br>Number of event workers that handle job-related events inside automation controller.                                                                                                                                                      | <br>Optional                                                 | <br>`4`                                                                        |
| <br>`controller_extra_settings`             | <br>Defines additional settings for use by automation controller during installation.<br>For example:   ``` controller_extra_settings=[{"setting": "USE_X_FORWARDED_HOST", "value": True}] ```                                                | <br>Optional                                                 | <br>`[]`                                                                       |
| <br>`controller_firewall_zone`              | <br>The firewall zone where automation controller related firewall rules are applied. This controls which networks can access automation controller based on the zone's trust level.                                                          | <br>Optional                                                 | <br>`public`                                                                   |
| <br>`controller_license_file`               | <br>Path to the automation controller license file.                                                                                                                                                                                           |                                                              |                                                                                |
| <br>`controller_nginx_client_max_body_size` | <br>Maximum allowed size for data sent to automation controller through NGINX.                                                                                                                                                                | <br>Optional                                                 | <br>`5m`                                                                       |
| <br>`controller_nginx_disable_hsts`         | <br>Controls whether HTTP Strict Transport Security (HSTS) is enabled or disabled for automation controller. Set this variable to `true` to disable HSTS.                                                                                     | <br>Optional                                                 | <br>`false`                                                                    |
| <br>`controller_nginx_disable_https`        | <br>Controls whether HTTPS is enabled or disabled for automation controller. Set this variable to `true` to disable HTTPS.                                                                                                                    | <br>Optional                                                 | <br>`false`                                                                    |
| <br>`controller_nginx_hsts_max_age`         | <br>Maximum duration (in seconds) that HTTP Strict Transport Security (HSTS) is enforced for automation controller.                                                                                                                           | <br>Optional                                                 | <br>`63072000`                                                                 |
| <br>`controller_nginx_http_port`            | <br>Port number that automation controller listens on for HTTP requests.                                                                                                                                                                      | <br>Optional                                                 | <br>`8080`                                                                     |
| <br>`controller_nginx_https_port`           | <br>Port number that automation controller listens on for HTTPS requests.                                                                                                                                                                     | <br>Optional                                                 | <br>`8443`                                                                     |
| <br>`controller_nginx_https_protocols`      | <br>Protocols that automation controller supports when handling HTTPS traffic.                                                                                                                                                                | <br>Optional                                                 | <br>`[TLSv1.2, TLSv1.3]`                                                       |
| <br>`controller_nginx_user_headers`         | <br>List of additional NGINX headers to add to automation controller's NGINX configuration.                                                                                                                                                   | <br>Optional                                                 | <br>`[]`                                                                       |
| <br>`controller_percent_memory_capacity`    | <br>Memory allocation for automation controller.                                                                                                                                                                                              | <br>Optional                                                 | <br>`1.0` (allocates 100% of the total system memory to automation controller) |
| <br>`controller_pg_cert_auth`               | <br>Controls whether client certificate authentication is enabled or disabled on the automation controller PostgreSQL database. Set this variable to `true` to enable client certificate authentication.                                      | <br>Optional                                                 | <br>`false`                                                                    |
| <br>`controller_pg_database`                | <br>Name of the PostgreSQL database used by automation controller.                                                                                                                                                                            | <br>Optional                                                 | <br>`awx`                                                                      |
| <br>`controller_pg_host`                    | <br>Hostname of the PostgreSQL database used by automation controller.                                                                                                                                                                        | <br>Required                                                 |                                                                                |
| <br>`controller_pg_password`                | <br>Password for the automation controller PostgreSQL database user. Use of special characters for this variable is limited. The `!`, `#`, `0` and `@` characters are supported. Use of other special characters can cause the setup to fail. | <br>Required if not using client certificate authentication. |                                                                                |
| <br>`controller_pg_port`                    | <br>Port number that automation controller uses to connect to its PostgreSQL database. This variable does not configure the PostgreSQL server listening port. To change the port the PostgreSQL server listens on, use `postgresql_port`.     | <br>Optional                                                 | <br>`5432`                                                                     |
| <br>`controller_pg_socket`                  | <br>UNIX socket used by automation controller to connect to the PostgreSQL database.                                                                                                                                                          | <br>Optional                                                 |                                                                                |
| <br>`controller_pg_sslmode`                 | <br>Controls the SSL/TLS mode to use when automation controller connects to the PostgreSQL database. Valid options include `verify-full`, `verify-ca`, `require`, `prefer`, `allow`, `disable`.                                               | <br>Optional                                                 | <br>`prefer`                                                                   |
| <br>`controller_pg_tls_cert`                | <br>Path to the PostgreSQL SSL/TLS certificate file for automation controller.                                                                                                                                                                | <br>Required if using client certificate authentication.     |                                                                                |
| <br>`controller_pg_tls_key`                 | <br>Path to the PostgreSQL SSL/TLS key file for automation controller.                                                                                                                                                                        | <br>Required if using client certificate authentication.     |                                                                                |
| <br>`controller_pg_username`                | <br>Username for the automation controller PostgreSQL database user.                                                                                                                                                                          | <br>Optional                                                 | <br>`awx`                                                                      |
| <br>`controller_secret_key`                 | <br>Secret key value used by automation controller to sign and encrypt data.                                                                                                                                                                  | <br>Optional                                                 |                                                                                |
| <br>`controller_tls_cert`                   | <br>Path to the SSL/TLS certificate file for automation controller.                                                                                                                                                                           | <br>Optional                                                 |                                                                                |
| <br>`controller_tls_key`                    | <br>Path to the SSL/TLS key file for automation controller.                                                                                                                                                                                   | <br>Optional                                                 |                                                                                |
| <br>`controller_tls_remote`                 | <br>Denote whether the automation controller provided certificate files are local to the installation program (`false`) or on the remote component server (`true`).                                                                           | <br>Optional                                                 | <br>`false`                                                                    |
| <br>`controller_use_archive_compression`    | <br>Controls whether archive compression is enabled or disabled for automation controller. You can control this functionality globally by using `use_archive_compression`.                                                                    | <br>Optional                                                 | <br>`true`                                                                     |
| <br>`controller_use_db_compression`         | <br>Controls whether database compression is enabled or disabled for automation controller. You can control this functionality globally by using `use_db_compression`.                                                                        | <br>Optional                                                 | <br>`true`                                                                     |
| <br>`controller_uwsgi_listen_queue_size`    | <br>Number of requests `uwsgi` allows in the queue on automation controller until `uwsgi_processes` can serve them.                                                                                                                           | <br>Optional                                                 | <br>`2048`                                                                     |
