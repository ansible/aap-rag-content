# Delete or unregister an execution environment definition

Remove an execution environment definition from the catalog in Ansible automation portal.

## Before you begin

- You have the AAP Administrator role, or you have been explicitly granted the Backstage catalog delete permission through RBAC.

## Procedure

1.  Open the execution environment definition detail page.
2.  Click the actions menu and select **Delete** (or **Unregister entity**).
3.  Confirm the deletion.
