# Automation hub variables

Reference information for the inventory file variables used to configure automation hub.

| Variable name                        | Description                                                                                                                                                                                                                                                                                                                                                                         | Required or optional                                                                                                                                                    | Default                                   |
| ------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------- |
| <br>`hub_admin_password`             | <br>Automation hub administrator password. Use of special characters for this variable is limited. The password can include any printable ASCII character except `/`, `"`, or `@`.                                                                                                                                                                                                  | <br>Required                                                                                                                                                            |                                           |
| <br>`hub_azure_account_key`          | <br>Azure blob storage account key.                                                                                                                                                                                                                                                                                                                                                 | <br>Required if using an Azure blob storage backend.                                                                                                                    |                                           |
| <br>`hub_azure_account_name`         | <br>Account name associated with the Azure blob storage.                                                                                                                                                                                                                                                                                                                            | <br>Required when using an Azure blob storage backend.                                                                                                                  |                                           |
| <br>`hub_azure_container`            | <br>Name of the Azure blob storage container.                                                                                                                                                                                                                                                                                                                                       | <br>Optional                                                                                                                                                            | <br>`pulp`                                |
| <br>`hub_azure_extra_settings`       | <br>Defines extra parameters for the Azure blob storage backend. For more information about the list of parameters, see [django-storages documentation - Azure Storage](https://django-storages.readthedocs.io/en/latest/backends/azure.html#settings).                                                                                                                             | <br>Optional                                                                                                                                                            | <br>`{}`                                  |
| <br>`hub_collection_auto_sign`       | <br>If a collection signing service is enabled, collections are not signed automatically by default. Set this variable to `true` to sign collections by default.                                                                                                                                                                                                                    | <br>Optional                                                                                                                                                            | <br>`false`                               |
| <br>`hub_collection_signing`         | <br>Set this variable to `true` to enable a collection signing service.                                                                                                                                                                                                                                                                                                             | <br>Optional                                                                                                                                                            | <br>`false`                               |
| <br>`hub_collection_signing_key`     | <br>Path to the collection signing key file.                                                                                                                                                                                                                                                                                                                                        | <br>Required if a collection signing service is enabled.                                                                                                                |                                           |
| <br>`hub_collection_signing_pass`    | <br>Password for the automation content collection signing service.                                                                                                                                                                                                                                                                                                                 | <br>Required if the collection signing service is protected by a passphrase.                                                                                            |                                           |
| <br>`hub_collection_signing_service` | <br>Service for signing collections.                                                                                                                                                                                                                                                                                                                                                | <br>Optional                                                                                                                                                            | <br>`ansible-default`                     |
| <br>`hub_container_signing`          | <br>Set this variable to `true` to enable a container signing service.                                                                                                                                                                                                                                                                                                              | <br>Optional                                                                                                                                                            | <br>`false`                               |
| <br>`hub_container_signing_key`      | <br>Path to the container signing key file.                                                                                                                                                                                                                                                                                                                                         | <br>Required if a container signing service is enabled.                                                                                                                 |                                           |
| <br>`hub_container_signing_pass`     | <br>Password for the automation content container signing service.                                                                                                                                                                                                                                                                                                                  | <br>Required if the container signing service is protected by a passphrase.                                                                                             |                                           |
| <br>`hub_container_signing_service`  | <br>Service for signing containers.                                                                                                                                                                                                                                                                                                                                                 | <br>Optional                                                                                                                                                            | <br>`container-default`                   |
| <br>`hub_data_path_exclude`          | <br>Automation hub backup path to exclude.                                                                                                                                                                                                                                                                                                                                          | <br>Optional                                                                                                                                                            | <br>`[]`                                  |
| <br>`hub_extra_settings`             | <br>Defines additional settings for use by automation hub during installation.<br>For example:   ``` hub_extra_settings=[{"setting": "REDIRECT_IS_HTTPS", "value": True}] ```                                                                                                                                                                                                       | <br>Optional                                                                                                                                                            | <br>`[]`                                  |
| <br>`hub_firewall_zone`              | <br>The firewall zone where automation hub related firewall rules are applied. This controls which networks can access automation hub based on the zone's trust level.                                                                                                                                                                                                              | <br>Optional                                                                                                                                                            | <br>`public`                              |
| <br>`hub_galaxy_importer`            | <br>Dictionary of settings to pass to the `galaxy-importer.cfg` configuration file. These settings control how the `galaxy-importer` service processes and validates Ansible content. Example values include: `ansible-doc`, `ansible-lint`, and `flake8`.                                                                                                                          | <br>Optional                                                                                                                                                            |                                           |
| <br>`hub_nginx_client_max_body_size` | <br>Maximum allowed size for data sent to automation hub through NGINX.                                                                                                                                                                                                                                                                                                             | <br>Optional                                                                                                                                                            | <br>`20m`                                 |
| <br>`hub_nginx_disable_hsts`         | <br>Controls whether HTTP Strict Transport Security (HSTS) is enabled or disabled for automation hub. Set this variable to `true` to disable HSTS.                                                                                                                                                                                                                                  | <br>Optional                                                                                                                                                            | <br>`false`                               |
| <br>`hub_nginx_disable_https`        | <br>Controls whether HTTPS is enabled or disabled for automation hub. Set this variable to `true` to disable HTTPS.                                                                                                                                                                                                                                                                 | <br>Optional                                                                                                                                                            | <br>`false`                               |
| <br>`hub_nginx_hsts_max_age`         | <br>Maximum duration (in seconds) that HTTP Strict Transport Security (HSTS) is enforced for automation hub.                                                                                                                                                                                                                                                                        | <br>Optional                                                                                                                                                            | <br>`63072000`                            |
| <br>`hub_nginx_http_port`            | <br>Port number that automation hub listens on for HTTP requests.                                                                                                                                                                                                                                                                                                                   | <br>Optional                                                                                                                                                            | <br>`8081`                                |
| <br>`hub_nginx_https_port`           | <br>Port number that automation hub listens on for HTTPS requests.                                                                                                                                                                                                                                                                                                                  | <br>Optional                                                                                                                                                            | <br>`8444`                                |
| <br>`hub_nginx_https_protocols`      | <br>Protocols that automation hub will support when handling HTTPS traffic.                                                                                                                                                                                                                                                                                                         | <br>Optional                                                                                                                                                            | <br>`[TLSv1.2, TLSv1.3]`                  |
| <br>`hub_nginx_user_headers`         | <br>List of additional NGINX headers to add to automation hub's NGINX configuration.                                                                                                                                                                                                                                                                                                | <br>Optional                                                                                                                                                            | <br>`[]`                                  |
| <br>`hub_pg_cert_auth`               | <br>Controls whether client certificate authentication is enabled or disabled on the automation hub PostgreSQL database. Set this variable to `true` to enable client certificate authentication.                                                                                                                                                                                   | <br>Optional                                                                                                                                                            | <br>`false`                               |
| <br>`hub_pg_database`                | <br>Name of the PostgreSQL database used by automation hub.                                                                                                                                                                                                                                                                                                                         | <br>Optional                                                                                                                                                            | <br>`pulp`                                |
| <br>`hub_pg_host`                    | <br>Hostname of the PostgreSQL database used by automation hub.                                                                                                                                                                                                                                                                                                                     | <br>Required                                                                                                                                                            |                                           |
| <br>`hub_pg_password`                | <br>Password for the automation hub PostgreSQL database user. Use of special characters for this variable is limited. The `!`, `#`, `0` and `@` characters are supported. Use of other special characters can cause the setup to fail.                                                                                                                                              | <br>Optional                                                                                                                                                            |                                           |
| <br>`hub_pg_port`                    | <br>Port number for the PostgreSQL database used by automation hub.                                                                                                                                                                                                                                                                                                                 | <br>Optional                                                                                                                                                            | <br>`5432`                                |
| <br>`hub_pg_socket`                  | <br>UNIX socket used by automation hub to connect to the PostgreSQL database.                                                                                                                                                                                                                                                                                                       | <br>Optional                                                                                                                                                            |                                           |
| <br>`hub_pg_sslmode`                 | <br>Controls the SSL/TLS mode to use when automation hub connects to the PostgreSQL database. Valid options include `verify-full`, `verify-ca`, `require`, `prefer`, `allow`, `disable`.                                                                                                                                                                                            | <br>Optional                                                                                                                                                            | <br>`prefer`                              |
| <br>`hub_pg_tls_cert`                | <br>Path to the PostgreSQL SSL/TLS certificate file for automation hub.                                                                                                                                                                                                                                                                                                             | <br>Required if using client certificate authentication.                                                                                                                |                                           |
| <br>`hub_pg_tls_key`                 | <br>Path to the PostgreSQL SSL/TLS key file for automation hub.                                                                                                                                                                                                                                                                                                                     | <br>Required if using client certificate authentication.                                                                                                                |                                           |
| <br>`hub_pg_username`                | <br>Username for the automation hub PostgreSQL database user.                                                                                                                                                                                                                                                                                                                       | <br>Optional                                                                                                                                                            | <br>`pulp`                                |
| <br>`hub_s3_access_key`              | <br>AWS S3 access key.                                                                                                                                                                                                                                                                                                                                                              | <br>Required if using an AWS S3 storage backend.                                                                                                                        |                                           |
| <br>`hub_s3_bucket_name`             | <br>Name of the AWS S3 storage bucket.                                                                                                                                                                                                                                                                                                                                              | <br>Optional                                                                                                                                                            | <br>`pulp`                                |
| <br>`hub_s3_extra_settings`          | <br>Used to define extra parameters for the AWS S3 storage backend. For more information about the list of parameters, see [django-storages documentation - Amazon S3](https://django-storages.readthedocs.io/en/latest/backends/amazon-S3.html#settings).                                                                                                                          | <br>Optional                                                                                                                                                            | <br>`{}`                                  |
| <br>`hub_s3_secret_key`              | <br>AWS S3 secret key.                                                                                                                                                                                                                                                                                                                                                              | <br>Required if using an AWS S3 storage backend.                                                                                                                        |                                           |
| <br>`hub_secret_key`                 | <br>Secret key value used by automation hub to sign and encrypt data.                                                                                                                                                                                                                                                                                                               | <br>Optional                                                                                                                                                            |                                           |
| <br>`hub_seed_collections`           | <br>Controls whether or not pre-loading of collections is enabled. When you run the bundle installer, validated content is uploaded to the `validated` repository, and certified content is uploaded to the `rh-certified` repository. By default, certified content and validated content are both uploaded. If you do not want to pre-load content, set this variable to `false`. | <br>Optional                                                                                                                                                            | <br>`true`                                |
| <br>`hub_shared_data_mount_opts`     | <br>Mount options for the Network File System (NFS) share.                                                                                                                                                                                                                                                                                                                          | <br>Optional                                                                                                                                                            | <br>`rw,sync,hard`                        |
| <br>`hub_shared_data_path`           | <br>Path to the Network File System (NFS) share with read, write, and execute (RWX) access. The value must match the format `host:dir`, for example `nfs-server.example.com:/exports/hub`.                                                                                                                                                                                          | <br>Required if installing more than one instance of automation hub with a `file` storage backend. When installing a single instance of automation hub, it is optional. |                                           |
| <br>`hub_storage_backend`            | <br>Automation hub storage backend type. Possible values include: `azure`, `file`, `s3`.                                                                                                                                                                                                                                                                                            | <br>Optional                                                                                                                                                            | <br>`file`                                |
| <br>`hub_tls_cert`                   | <br>Path to the SSL/TLS certificate file for automation hub.                                                                                                                                                                                                                                                                                                                        | <br>Optional                                                                                                                                                            |                                           |
| <br>`hub_tls_key`                    | <br>Path to the SSL/TLS key file for automation hub.                                                                                                                                                                                                                                                                                                                                | <br>Optional                                                                                                                                                            |                                           |
| <br>`hub_tls_remote`                 | <br>Denote whether the automation hub provided certificate files are local to the installation program (`false`) or on the remote component server (`true`).                                                                                                                                                                                                                        | <br>Optional                                                                                                                                                            | <br>`false`                               |
| <br>`hub_use_archive_compression`    | <br>Controls whether archive compression is enabled or disabled for automation hub. You can control this functionality globally by using `use_archive_compression`.                                                                                                                                                                                                                 | <br>Optional                                                                                                                                                            | <br>`true`                                |
| <br>`hub_use_db_compression`         | <br>Controls whether database compression is enabled or disabled for automation hub. You can control this functionality globally by using `use_db_compression`.                                                                                                                                                                                                                     | <br>Optional                                                                                                                                                            | <br>`true`                                |
| <br>`hub_workers`                    | <br>Number of automation hub workers.                                                                                                                                                                                                                                                                                                                                               | <br>Optional                                                                                                                                                            | <br>`2`                                   |
| <br>`--`                             | <br>Gunicorn worker timeout in seconds for automation hub. Controls how long a Gunicorn worker can process a single request before it is terminated. Computed from `client_request_timeout`.                                                                                                                                                                                        | <br>Optional                                                                                                                                                            | <br>`max(client_request_timeout, 10) / 3` |
| <br>`hub_nginx_read_timeout`         | <br>NGINX `proxy_read_timeout` value in seconds for automation hub. Controls how long NGINX waits to receive a response from the upstream Gunicorn server. Computed from `client_request_timeout`.                                                                                                                                                                                  | <br>Optional                                                                                                                                                            | <br>`max(client_request_timeout, 10) / 2` |
| <br>`hub_nginx_send_timeout`         | <br>NGINX `proxy_send_timeout`value in seconds for automation hub. Controls how long NGINX waits to transmit a request to the upstream Gunicorn server. Computed from `client_request_timeout`.                                                                                                                                                                                     | <br>Optional                                                                                                                                                            | <br>`max(client_request_timeout, 10) / 2` |
