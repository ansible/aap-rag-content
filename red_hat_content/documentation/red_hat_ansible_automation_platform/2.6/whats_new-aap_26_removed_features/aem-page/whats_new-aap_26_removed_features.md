# Removed features

Removed features are those that were deprecated in earlier releases. They are now removed from the Ansible Automation Platform 2.5, and will no longer be supported.

| Component                           | Feature                                                                                               |
| ----------------------------------- | ----------------------------------------------------------------------------------------------------- |
| <br>Event-Driven Ansible controller | <br>Removal of `max_running_activations setting in eda-controller`                                    |
| <br>Platform gateway                | <br>Legacy Authenticators that were added during an upgrade from 2.4 to 2.5 will no longer be present |
