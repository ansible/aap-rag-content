# Supported RHEL versions by deployment type

Supported RHEL versions differ among deployment types, as shown in the following table.

| Deployment type and version          | Supported RHEL version                                                                                                                                                      |
| ------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <br>RPM 2.6                          | <br>RHEL 9                                                                                                                                                                  |
| <br>Containerized 2.6                | <br>RHEL 9, RHEL 10                                                                                                                                                         |
| <br>OpenShift Container Platform 2.6 | <br>For RHEL versions included with OpenShift Container Platform, see [Red Hat OpenShift Container Platform Life Cycle Policy](https://access.redhat.com/articles/6907891). |
