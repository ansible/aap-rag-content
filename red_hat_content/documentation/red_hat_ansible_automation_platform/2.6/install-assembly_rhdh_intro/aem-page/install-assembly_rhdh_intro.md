# Install Ansible plug-ins for Red Hat Developer Hub (Red Hat OpenShift Container Platform only)

Ansible plug-ins for Red Hat Developer Hub (RHDH) offer a tailored Red Hat Developer Hub interface, curated learning paths, and software templates for creating projects, all while linking to supported development environments like OpenShift Dev Spaces and Ansible Automation Platform.
