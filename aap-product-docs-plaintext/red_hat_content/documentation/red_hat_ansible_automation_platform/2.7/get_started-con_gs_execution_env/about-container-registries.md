# Create automation runtimes with execution and decision environments
## About container registries

If you have many automation execution environments that you want to support, you can store them in a container registry linked to your private automation hub.

For more information, see [Populating your private automation hub container registry](/documentation/en-us/red_hat_ansible_automation_platform/2.7/administer-con_container_registry#populate-container-registry "The automation hub remote registry is used for storing and managing automation execution environments.").
