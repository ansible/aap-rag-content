# Review the feature flag display

Superusers and auditors can view additional information about available feature flags.

The Feature Flags UI displays the following information for each flag:

- **Display name**: The feature flag name.
- **Description**: The feature flag's purpose.
- **Support level**: Can be technology preview or developer preview.
- **State**: Can be either **enabled**or **disabled**. Current state information is visible to auditors only.
- **Toggle control**: Available only to superusers. For install-time feature flags, the toggle is visible to superusers but read-only.
- **Labels**: Indicate which Ansible service uses the flag (for example, controller or gateway).
- **Support**: Links to documentation about the specific feature flag.
You can filter feature flags in the toolbar by name, support level, or labels.
