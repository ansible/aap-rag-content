# Configure third-party services

The logging aggregator service works with the following monitoring and data analysis systems:

-  [Splunk](https://docs.redhat.com/en/documentation/red_hat_ansible_automation_platform/2.5/html/configuring_automation_execution/assembly-controller-logging-aggregation#ref-controller-logging-splunk)
-  [Loggly](https://docs.redhat.com/en/documentation/red_hat_ansible_automation_platform/2.5/html/configuring_automation_execution/assembly-controller-logging-aggregation#ref-controller-logging-loggly)
-  [Sumologic](https://docs.redhat.com/en/documentation/red_hat_ansible_automation_platform/2.5/html/configuring_automation_execution/assembly-controller-logging-aggregation#ref-controller-logging-sumologic)
-  [Elastic Stack (formerly ELK stack)](https://docs.redhat.com/en/documentation/red_hat_ansible_automation_platform/2.5/html/configuring_automation_execution/assembly-controller-logging-aggregation#ref-controller-logging-elastic-stack)
