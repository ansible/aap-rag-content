# View, create, or assign roles to users
## Reset a user password

As a platform administrator, you can reset local user passwords in platform gateway to restore access for users who are locked out or have forgotten their credentials.

### Before you begin

- You have platform administrator access.

### Procedure

1.  From the navigation panel, select Access Management> (and then)Users.
2.  Click the Edit icon next to the user you want to edit and select Edit user.
3.  In the **Password** field, enter a new password.
4.  Click Save user.
5.  Securely notify the user of the new password.

### Results

Confirm the user can log in to platform gateway by using the new password.
