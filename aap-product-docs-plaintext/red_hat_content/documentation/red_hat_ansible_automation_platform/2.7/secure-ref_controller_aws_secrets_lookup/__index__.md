# AWS secrets manager lookup

Use this credential type so that automation controller can retrieve secrets from AWS Secrets Manager.

This is part of the secret management capability. For more information, see [AWS Secrets Manager lookup](/documentation/en-us/red_hat_ansible_automation_platform/2.7/secure-integrate_third_party_secret_management_systems#ref-aws-secrets-manager-lookup "This plugin enables Amazon Web Services to be used as a credential input source to pull secrets from the Amazon Web Services Secrets Manager. The AWS Secrets Manager provides similar service to Microsoft Azure Key Vault, and the AWS collection provides a lookup plugin for it.")
