# View feature flags in your user interface
## Before you begin

The `RUNTIME_FEATURE_FLAGS` system setting must be enabled (set to `True`) for feature flags to be viewable in the UI.

## Procedure

1.  Log in to Ansible Automation Platform.
2.  From the navigation menu, go to Settings> (and then)Feature Flags.
3.  The Feature Flags page displays all available feature flags.
