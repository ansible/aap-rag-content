# 10. Patch releases
## 10.7. Ansible Automation Platform patch release May 28, 2025
### 10.7.5. Known Issues




- Red Hat Ansible Lightspeed enabled deployments must apply a workaround to avoid problems during upgrade from release 2.5.20250507. The service cluster and related objects must be removed before upgrade and re-created after upgrade. For more information please see this [KCS article](https://access.redhat.com/articles/7122651) .(AAP-46154)
