# 3. Managed Node Configuration
## 3.1. Red Hat Enterprise Linux managed node configuration




The following section provides guidance for _Red Hat Enterprise Linux_ (RHEL) managed nodes, but the concepts may be applicable to other Linux distributions as well.

Examples are provided for manual configuration of RHEL managed nodes. These steps can also be automated with Ansible Automation Platform, or they can be added to a _Standard Operating Environment_ (SOE) or "golden image" created with a tool such as the Red Hat Insights image builder.
