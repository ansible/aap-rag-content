# 7. Target environment
## 7.2. OpenShift Container Platform
### 7.2.4. Validating the target environment




To validate your migrated environment, perform the following steps.

**Procedure**

1. Verify all migrated components are functional.
1. Test workflows and automation processes.
1. Validate user access and permissions.
1. Confirm content synchronization and availability.
1. Test integration with OpenShift Container Platform-specific features.
