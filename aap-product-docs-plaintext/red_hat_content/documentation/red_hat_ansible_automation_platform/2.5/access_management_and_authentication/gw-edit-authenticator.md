# 3. Configuring authentication in the Ansible Automation Platform
## 3.7. Managing authentication in Ansible Automation Platform
### 3.7.4. Editing an authenticator




You can modify the settings of previously configured authenticators from the **Authentication** list view.

**Procedure**

1. From the navigation panel, selectAccess Management→Authentication Methods.
1. In the list view, you can either:


1. Select theEdit![Edit](https://access.redhat.com/webassets/avalon/d/Red_Hat_Ansible_Automation_Platform-2.5-Access_management_and_authentication-en-US/images/e5cb7f9b0cfe60ab5ba421bd17ecbb6a/leftpencil.png)
icon next to authenticator you want to modify, or
1. Select the authenticator name displayed in the **Name** column and clickEdit authenticatorfrom the **Details** page.

1. Modify the authentication details or mapping configurations as required.
1. ClickSave.
