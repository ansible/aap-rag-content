# 5. Managing access with role based access control
## 5.4. Resources
### 5.4.2. Providing user access to a resource




You can grant users access to resources through the roles to which they are assigned.

Note
Direct user access cannot be granted toAutomation Content→Remote Registriesresources.



**Procedure**

1. From the navigation panel, select a resource to which you want to provide team access. For example,Automation Execution→Templates.
1. Select the **User access** tab.
1. ClickAdd roles.
1. Click the checkbox beside the user to assign that user to your chosen type of resource and clickNext.
1. Select the roles you want applied to the user for the chosen resource and clickNext.
1. Review the settings and clickFinish. The Add roles dialog displays indicating whether the role assignments were successfully applied.
1. You can remove resource access for a user by selecting the **Remove role** icon next to the user. This launches a confirmation dialog, asking you to confirm the removal.
