# 2. Configuring authentication in the Ansible Automation Platform
## 2.7. Managing authentication in Ansible Automation Platform
### 2.7.3. Displaying authenticator details




After you locate the authenticator you want to review, you can display the configuration details:

**Procedure**

1. From the navigation panel, selectAccess Management→Authentication Methods.
1. In the list view, select the authenticator name displayed in the **Name** column.

The authenticator **Details** page is displayed.


1. From the **Details** page, you can review the configuration settings applied to the authenticator.
