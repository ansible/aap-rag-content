# 6. Roles
## 6.4. Deleting a role




Built in roles can not be deleted, however, you can delete custom roles from the **Roles** list view.

**Procedure**

1. From the navigation panel, selectAccess Management→Roles.
1. Select a tab for the component resource for which you want to create custom roles.

Note
If you have multiple Ansible Automation Platform components installed, you will see selections for the roles associated with each component in the **Roles** menu bar. For example, Automation Execution for automation controller roles, Automation Decisions for Event-Driven Ansible roles.




1. Click the **More Actions** icon **⋮** next to the role you want and select **Delete role** .
1. To delete roles in bulk, select the roles you want to delete from the **Roles** list view, click the **More Actions** icon **⋮** , and select **Delete roles** .
