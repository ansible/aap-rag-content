# 6. Upgrading Red Hat Ansible Automation Platform Operator on Red Hat OpenShift Container Platform
## 6.2. Upgrade considerations




If you are upgrading from version 2.4, continue to the [Upgrading the Ansible Automation Platform Operator](https://docs.redhat.com/en/documentation/red_hat_ansible_automation_platform/2.5/html-single/installing_on_openshift_container_platform/index#upgrading-operator_operator-upgrade) .

If your OpenShift Container Platform version is not supported by the Red Hat Ansible Automation Platform version you are upgrading to, you must upgrade your OpenShift Container Platform cluster to a supported version first.

Refer to the [Red Hat Ansible Automation Platform Life Cycle](https://access.redhat.com/support/policy/updates/ansible-automation-platform) to determine the OpenShift Container Platform version needed.

For information about upgrading your cluster, refer to [Updating clusters](https://docs.redhat.com/en/documentation/openshift_container_platform/4.7/html-single/updating_clusters/index) .
