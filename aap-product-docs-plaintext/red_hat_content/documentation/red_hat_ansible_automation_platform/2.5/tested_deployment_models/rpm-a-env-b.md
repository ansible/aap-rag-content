# 2. RPM topologies
## 2.2. RPM mixed growth topology




The growth topology is intended for organizations that are getting started with Ansible Automation Platform and do not require redundancy or higher compute for large volumes of automation. This topology allows for smaller footprint deployments. The mixed topology has different versions of Ansible Automation Platform intended for configuring a new installation of Event-Driven Ansible 1.1 with automation controller 4.4 or 4.5.
