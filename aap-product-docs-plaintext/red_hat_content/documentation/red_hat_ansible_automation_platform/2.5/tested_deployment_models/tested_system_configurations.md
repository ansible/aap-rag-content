| Type | Description |
| --- | --- |

- Red Hat Enterprise Linux 9.2 or later minor versions of Red Hat Enterprise Linux 9. |
| CPU architecture | x86_64, AArch64, s390x (IBM Z), ppc64le (IBM Power) |
| Ansible-core | Ansible-core version 2.16 or later |
