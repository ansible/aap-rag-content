# 3. Pre-installation configuration
## 3.3. Generating GitHub and Gitlab personal access tokens
### 3.3.2. Creating a Personal access token (PAT) on Gitlab




1. In a browser, log in to Gitlab and navigate to the [Personal access tokens](https://gitlab.com/-/user_settings/personal_access_tokens) page.
1. Click **Add new token** .
1. Provide a name and expiration date for the token.
1. In the **Scopes:** section, select the following:


- read_repository
- api

1. Click **Create personal access token** .
1. Save the Personal access token.
