# 3. Pre-installation configuration
## 3.3. Generating GitHub and Gitlab personal access tokens
### 3.3.1. Creating a Personal access token (PAT) on GitHub




1. In a browser, log in to GitHub and navigate to the [Personal access tokens](https://github.com/settings/tokens) page.
1. Click **Generate new token(classic)** .
1. In the **Select scopes:** section, enable the following:


- repo
- read:org
- workflow (as needed)

1. Click **Generate token** .
1. Save the Personal access token.
