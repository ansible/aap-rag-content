# Chapter 5. Automation Analytics Data Dictionary




Automation Analytics Data is sent to the Red Hat Hybrid Cloud Console (HCC) to provide detailed analytics on your automation.

The [Ansible Automation Platform - Data Dictionary](https://access.redhat.com/articles/7124201) knowledge base article outlines the data dictionary for the information collected by Automation Analytics from the Red Hat Ansible Automation Platform automation controller, also known as Automation Execution.
