# 2. Workflow for developing automation content
## 2.1. Workflow
### 2.1.2. Test




1. Debug your playbook with the help of `    ansible-lint` .
1. Select or create an automation execution environment so that your local environment replicates the environment on Ansible Automation Platform.
1. Run your playbooks from VS Code, using `    ansible-playbook` or using `    ansible-navigator` with an execution environment.
1. Test your playbooks by running them on an execution environment that replicates your production environment.
