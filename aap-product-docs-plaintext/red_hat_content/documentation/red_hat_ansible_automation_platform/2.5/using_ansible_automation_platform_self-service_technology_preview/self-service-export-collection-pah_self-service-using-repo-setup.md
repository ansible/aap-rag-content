# 3. Setting up repositories for collections
## 3.1. Exporting a collection from private automation hub




1. Log in to Ansible Automation Platform.
1. From the navigation panel, selectAutomation Content→Collections.

The **Collections** page displays all collections across all repositories. You can search for a specific collection.


1. Click into the collection that you want to export.

The **Collection details** page opens.


1. From the **Install** tab, select **Download tarball** .

The `    .tar` file is downloaded to your default browser downloads folder.
