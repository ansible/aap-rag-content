# Chapter 1. Overview




To populate self-service technology preview with templates, you must create repositories in GitHub or Gitlab for collections that define the templates.

Currently, Red Hat provides validated content with examples to automate jobs for RHEL, Network, Windows, and Cloud using self-service technology preview.

Note
The validated content examples provided have their own support statement and lifecycle definitions. Refer to the [Red Hat Ansible Automation Platform Life Cycle](https://access.redhat.com/support/policy/updates/ansible-automation-platform#validated) .



These collections are hosted in automation hub. Your administrator must add the collections to private automation hub so that you can download them and create the repositories.

You can then import templates into your self-service technology preview, and launch them to run automation jobs.
