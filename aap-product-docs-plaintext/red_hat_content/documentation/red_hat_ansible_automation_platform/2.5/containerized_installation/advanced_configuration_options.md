# 2. Ansible Automation Platform containerized installation
## 2.7. Advanced configuration options




Advanced configuration options, such as external database set up and the use of custom TLS certs, are available for more complex deployments of containerized Ansible Automation Platform.

If you are not using these advanced configuration options, go to [Installing containerized Ansible Automation Platform](https://docs.redhat.com/en/documentation/red_hat_ansible_automation_platform/2.5/html/containerized_installation/aap-containerized-installation#installing-containerized-aap) to continue with your installation.
