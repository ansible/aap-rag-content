# 4. Getting started as an automation operator
## 4.7. Automation execution projects
### 4.7.1. Viewing project details




The **Projects** page displays a list of projects that are currently available.

1. From the navigation panel, selectAutomation Execution→Projects.
1. Click a project to view its details.
1. For each project listed, you can sync the latest revision, edit the project, or copy the project’s attributes using the icons next to each project.
