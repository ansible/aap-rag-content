# 4. Projects
## 4.4. Deleting a project




If you need to delete a project, the Event-Driven Ansible controller interface provides multiple options.

**Procedure**

1. To delete a project, complete one of the following:


- From the **Projects** list view, select the checkbox next to the desired project, and click theMore Actionsicon **⋮** from the page menu.
- From the **Projects** list view, click theMore Actionsicon **⋮** next to the desired project.

1. SelectDelete project.
1. In the **Permanently delete projects** window, selectYes, I confirm that I want to delete this project.
1. SelectDelete project.
