# Chapter 4. Deprovisioning individual nodes or groups




You can deprovision automation mesh nodes and instance groups by using the Ansible Automation Platform installer. Learn how to deprovision specific nodes or entire groups, with example inventory files.

Important
Containerized Ansible Automation Platform deployments do not support the `node_state=deprovision` parameter. For containerized deployments, use the `awx-manage` command to deprovision nodes and groups. For more information, see [Deprovisioning isolated nodes](https://docs.redhat.com/en/documentation/red_hat_ansible_automation_platform/2.5/html/automation_mesh_for_vm_environments/assembly-deprovisioning-mesh#proc-deprovision-isolated-nodes) and [Deprovisioning isolated instance groups](https://docs.redhat.com/en/documentation/red_hat_ansible_automation_platform/2.5/html/automation_mesh_for_vm_environments/assembly-deprovisioning-mesh#proc-deprovision-isolated-groups) .
