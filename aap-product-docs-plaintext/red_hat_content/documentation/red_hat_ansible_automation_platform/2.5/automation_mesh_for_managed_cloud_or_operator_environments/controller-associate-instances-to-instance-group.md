# 2. Automation mesh for operator-based Red Hat Ansible Automation Platform
## 2.5. Associating instances to an instance group




**Procedure**

1. Select the **Instances** tab on the **Details** page of an Instance Group.
1. ClickAssociate instance.
1. Click the checkbox next to one or more available instances from the list to select the instances you want to associate with the instance group and clickConfirm
