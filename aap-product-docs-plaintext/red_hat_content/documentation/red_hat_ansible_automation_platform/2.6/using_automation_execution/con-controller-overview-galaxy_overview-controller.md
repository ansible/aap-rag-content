# 1. Automation controller overview
## 1.7. Ansible Galaxy integration




Automation controller integrates with Ansible Galaxy to simplify the process of using and managing Ansible roles in your automation projects.

By including an Ansible Galaxy `requirements.yml` file in your project directory, automation controller automatically fetches the roles your playbook needs from Galaxy, GitHub, or your local source control. For more information, see [Ansible Galaxy Support](https://docs.redhat.com/en/documentation/red_hat_ansible_automation_platform/2.6/html/using_automation_execution/controller-projects#ref-projects-galaxy-support) .
