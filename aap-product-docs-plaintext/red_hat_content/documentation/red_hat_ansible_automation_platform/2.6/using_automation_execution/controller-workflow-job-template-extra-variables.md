# 8. Workflow job templates
## 8.10. Workflow job template extra variables




You can define extra variables for a workflow job template that are passed to all job templates within the workflow.

For more information see the [Extra variables](https://docs.redhat.com/en/documentation/red_hat_ansible_automation_platform/2.6/html/using_automation_execution/controller-job-templates#controller-extra-variables) section.
