# 1. Automation controller overview
## 1.12. Integrations

Automation controller supports the following integrations:

- Dynamic inventory sources for Red Hat Satellite 6.

For more information, see [Red Hat Satellite 6](#proc-controller-inv-source-satellite "14.4.7.7.&nbsp;Red Hat Satellite 6").

- Red Hat Lightspeed integration, enabling Red Hat Lightspeed playbooks to be used as an Ansible Automation Platform project.

For more information, see [Setting up Red Hat Lightspeed for Red Hat Ansible Automation Platform Remediations](#controller-setting-up-insights "Chapter&nbsp;28.&nbsp;Setting up Red Hat Lightspeed Remediations").

- Automation hub acts as a content provider for automation controller, requiring both an automation controller deployment and an automation hub deployment running alongside each other.
