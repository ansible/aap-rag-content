## 4.2. Sort




Automation controller provides sorting capabilities in various lists and tables throughout the user interface. To sort a list or table, click the column header you want to sort by.

Clicking the header once sorts the list in ascending order; clicking it again sorts it in descending order.

The following is an example from the schedules list:

![sort arrow](https://access.redhat.com/webassets/avalon/d/Red_Hat_Ansible_Automation_Platform-2.6-Using_automation_execution-en-US/images/6f66263442890fd4e275490c7cc6d7bc/sort-order-example.png)


The direction of the arrow indicates the sort order of the column.
