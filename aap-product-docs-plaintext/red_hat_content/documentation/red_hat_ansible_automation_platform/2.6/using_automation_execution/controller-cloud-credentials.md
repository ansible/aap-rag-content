# 6. Job templates
## 6.15. Use Cloud Credentials with a cloud inventory

Automation controller can use Cloud Credentials to authenticate to cloud providers.

Cloud Credentials can be used when syncing a cloud inventory. They can also be associated with a job template and included in the runtime environment for use by a playbook. The following Cloud Credentials are supported:

- [Openstack](#controller-openstack-cloud "6.16.&nbsp;OpenStack")
- [Amazon Web Services](#controller-aws-cloud "6.16.1.&nbsp;Amazon Web Services")
- [Google](#controller-google-cloud "6.16.2.&nbsp;Google")
- [Azure](#controller-azure-cloud "6.16.3.&nbsp;Azure")
- [VMware](#controller-vmware-cloud "6.16.4.&nbsp;VMware")
