# 1. Automation controller overview
## 1.5. The ideal RESTful API




The automation controller REST API is the ideal RESTful API for a systems management application, with all resources fully discoverable, paginated, searchable, and well modeled.

A styled API browser enables API exploration from the API root at `http://&lt;server name&gt;/api/` , showing off every resource and relation. Everything that can be done in the user interface can be done in the API.
