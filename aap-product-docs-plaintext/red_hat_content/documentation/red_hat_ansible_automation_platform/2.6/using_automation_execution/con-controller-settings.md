# 3. The User Interface
## 3.3. The Settings menu




You can configure some automation controller options by using the **Settings** menu of the User Interface.

The **Settings** page enables an administrator to configure the following:

-  [Configuring subscriptions](https://docs.redhat.com/en/documentation/red_hat_ansible_automation_platform/2.6/html/access_management_and_authentication/assembly-gw-settings#proc-controller-configure-subscriptions)
-  [Platform gateway](https://docs.redhat.com/en/documentation/red_hat_ansible_automation_platform/2.6/html/access_management_and_authentication/assembly-gw-settings#proc-settings-platform-gateway)
-  [User preferences](https://docs.redhat.com/en/documentation/red_hat_ansible_automation_platform/2.6/html/access_management_and_authentication/assembly-gw-settings#proc-settings-user-preferences)
-  [Configuring jobs](https://docs.redhat.com/en/documentation/red_hat_ansible_automation_platform/2.6/html/configuring_automation_execution/controller-config#controller-configure-jobs)
-  [Setting up logging](https://docs.redhat.com/en/documentation/red_hat_ansible_automation_platform/2.6/html/configuring_automation_execution/assembly-controller-logging-aggregation#proc-controller-set-up-logging)
-  [Troubleshooting options](https://docs.redhat.com/en/documentation/red_hat_ansible_automation_platform/2.6/html/access_management_and_authentication/assembly-gw-settings#proc-settings-troubleshooting)
