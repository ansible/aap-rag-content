# 3. Manage containers in private automation hub
## 3.6. Working with signed containers
### 3.6.5. Using Podman to ensure an image is signed by a specific signature




Podman and other image clients can use policies to ensure the validity of an image. To enable this capability, assign a policy to the signature.

To ensure an execution environment is signed by specific signatures, the signatures must first be on your local environment.

**Procedure**

1. From the navigation panel, selectAutomation Content→Signature Keys.
1. Click theDownload keyicon next to the signature that you are using. A new window should open to indicate you have downloaded the key.
