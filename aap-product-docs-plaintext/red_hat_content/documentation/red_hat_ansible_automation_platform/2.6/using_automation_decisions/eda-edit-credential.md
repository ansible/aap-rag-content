# 2. Credentials
## 2.3. Editing a credential




Edit credentials to update expired tokens, rotate secrets, or adjust permissions, ensuring secure and continuous access to external systems.

**Procedure**

1. Edit the credential by using one of these methods:


- From the **Credentials** list view, click theEdit credentialicon next to the desired credential.
- From the **Credentials** list view, select the name of the credential, clickEdit credential.

1. Edit the appropriate details and clickSave credential.
