# 10. Simplified event routing
## 10.9. Resending webhook data from your event stream type




After you have replaced your sources with the event stream you created, resend previous webhook data from the event stream to efficiently test and verify that the new routing successfully triggers the rulebook activation.

**Procedure**

1. Go back to the **GitHub Webhook / Manage webhook** page.
1. Click the **Recent Deliveries** tab.
1. Click theellipsis.
1. ClickRedeliver. A **Redeliver payload?** window is displayed with a delivery message.
1. Click **Yes, redeliver this payload** .
1. Return to the Ansible Automation Platform to check your rule audit.
