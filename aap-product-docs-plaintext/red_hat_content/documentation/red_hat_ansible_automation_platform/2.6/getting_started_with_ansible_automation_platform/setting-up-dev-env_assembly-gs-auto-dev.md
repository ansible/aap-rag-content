# 3. Getting started as an automation developer
## 3.1. Setting up your development environment




Before you begin to create content, consult our guide to [Developing automation content](https://docs.redhat.com/en/documentation/red_hat_ansible_automation_platform/2.6/html/developing_automation_content) . There you can find information on Ansible development tools, which you can integrate into your environment, and learn how to scaffold a playbook project.
