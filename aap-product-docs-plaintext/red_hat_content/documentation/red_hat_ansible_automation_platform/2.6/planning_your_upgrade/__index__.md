## Planning for a successful upgrade of Ansible Automation Platform

**Abstract**

This guide provides a detailed overview of the changes and improvements in the upgrade process from Ansible Automation Platform 2.4 to 2.6, including new infrastructure requirements, data migration specifics, and a comparison of different deployment types. It addresses frequently asked questions and outlines how existing authentication and user data are handled during the upgrade.
