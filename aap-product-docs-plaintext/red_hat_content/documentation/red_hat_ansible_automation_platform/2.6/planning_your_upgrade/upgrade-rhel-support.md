# 3. Support matrix for upgrade scenarios
## 3.1. Supported RHEL versions by deployment type




Supported RHEL versions differ among deployment types, as shown in the following table.

| Deployment type and version | Supported RHEL version |
| --- | --- |
| RPM 2.6 | RHEL 9 |
| Containerized 2.6 | RHEL 9, RHEL 10 |
| OpenShift Container Platform 2.6 | For RHEL versions included with OpenShift Container Platform, see [Red Hat OpenShift Container Platform Life Cycle Policy](https://access.redhat.com/articles/6907891) . |
