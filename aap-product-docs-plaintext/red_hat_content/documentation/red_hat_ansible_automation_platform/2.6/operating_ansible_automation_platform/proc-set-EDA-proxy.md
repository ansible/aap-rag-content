# 3. Configuring Ansible Automation Platform to use egress proxy
## 3.6. Configuring proxy settings on Event-Driven Ansible




For Event-Driven Ansible, there are no global settings to set a proxy. You must specify the proxy for every project.

**Procedure**

1. From the navigation panel, selectAutomation Decisions→Projects.
1. ClickCreate Project

Use the **Proxy** field.
