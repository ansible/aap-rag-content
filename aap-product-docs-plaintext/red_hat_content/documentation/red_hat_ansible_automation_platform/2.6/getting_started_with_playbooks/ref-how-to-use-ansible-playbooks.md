# 1. Introduction
## 1.2. How do I use Ansible Playbooks?




Ansible uses the YAML syntax. YAML is a human-readable language that enables you to create playbooks without having to learn a complicated coding language.

For more information on YAML, see YAML Syntax and consider installing an add-on for your text editor, see [Other Tools and Programs](https://docs.ansible.com/ansible/latest/community/other_tools_and_programs.html#other-tools-and-programs) to help you write clean YAML syntax in your playbooks.

There are two ways of using Ansible Playbooks:

- From the _command line interface_ (CLI)
- Using Red Hat Ansible Automation Platform’s push-button deployments.
