# 6. Create and integrate custom MCP Servers in execution environments
## 6.10. Use your MCP-enabled execution environment in Ansible Automation Platform
### 6.10.3. Assign to a Job Template

To execute automation tasks using your custom tools, assign the MCP-enabled execution environment to a job template and configure any necessary server credentials.

**Procedure**

1. In the navigation panel, select Automation Execution → Templates.
2. Edit or create a Job Template.
3. In the **Execution Environment** field, select your MCP-enabled execution environment.
4. Configure any required credentials for the MCP server (such as AWS credentials).
5. Save and launch the template.
