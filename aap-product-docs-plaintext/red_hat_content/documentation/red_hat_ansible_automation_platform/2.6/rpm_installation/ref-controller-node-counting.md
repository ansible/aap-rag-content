# 1. Red Hat Ansible Automation Platform installation overview
## 1.2. Managing Ansible Automation Platform subscriptions, updates, and support
### 1.2.2. Node counting in subscriptions




The Ansible Automation Platform subscription defines the number of Managed Nodes that can be managed as part of your subscription.

For more information about managed node requirements for subscriptions, see [How are "managed nodes" defined as part of the Red Hat Ansible Automation Platform offering](https://access.redhat.com/articles/3331481) .

Note
Ansible does not recycle node counts or reset automated hosts.
