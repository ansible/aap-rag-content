# 4. Managing access with role-based access control
## 4.3. Users
### 4.3.4. Deleting a user




Before you can delete a user, you must have normal user or system administrator permissions. When you delete a user account, the name and email of the user are permanently removed from Ansible Automation Platform.

**Procedure**

1. From the navigation panel, selectAccess Management→Users.
1. To delete a single user, select theMore Actions **⋮** icon next to the user you want to remove and select **Delete user** .
1. To bulk delete users, select the checkbox next to each user you want to remove, and then from theMore Actions **⋮** list, click **Delete users** .
