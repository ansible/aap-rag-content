# 2. Configuring authentication in the Ansible Automation Platform
## 2.3. Creating an authentication method

Creating an authenticator involves the following procedures:

- [Selecting an authentication type](#gw-select-auth-type "2.3.1.&nbsp;Selecting an authentication type"), where you select the type of authenticator plugin you want to configure, including the authentication details for the authentication type selected.

- [Mapping](#gw-define-rules-triggers "2.3.2.&nbsp;Defining authentication mapping rules and triggers"), where you define mapping rule types and triggers to control access to the system, and [Mapping order](#gw-adjust-mapping-order "2.3.3.&nbsp;Adjusting the Mapping order"), where you can define the mapping precedence.


Note
Mapping order is only available if you have defined one or more authenticator maps.
