# 4. Managing access with role-based access control
## 4.2. Teams
### 4.2.5. Assigning administrators to a team




You can assign administrators to a team, which allows them to manage the membership and settings of that team. For example, they can create new users and grant permission to users within the team. To assign an administrator to a team, the administrator must already have been created. For more information, see [Creating a user](https://docs.redhat.com/en/documentation/red_hat_ansible_automation_platform/2.6/html/access_management_and_authentication/gw-managing-access#proc-controller-creating-a-user) .

**Procedure**

1. From the navigation panel, selectAccess Management→Teams.
1. Select the team to which you want to add an administrator.
1. Select the **Administrators** tab and clickAdd administrator(s).
1. Select one or more users from the list by clicking the checkbox next to the name to add them as administrators of this team.
1. ClickAdd administrators.
