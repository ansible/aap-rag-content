# 3. Deploying Red Hat Ansible Automation Platform on Microsoft Azure
## 3.1. Locating Ansible Automation Platform in Azure Marketplace




Before you can start provisioning your Ansible Automation Platform on Microsoft Azure, you must find the offering on the Azure marketplace, enable your subscription, and initiate a deployment.

**Procedure**

1. In a browser, navigate to the Azure Marketplace.
1. Click **Private Products** from the menu on the left of the screen.
1. Search for "Red Hat Ansible Automation Platform".
1. Click the card that is returned in the search. Be sure to select the official offering from Red Hat.
1. ClickGet it Now.
1. ClickContinue.
1. The **Overview** tab contains important information about activating your subscription for Ansible Automation Platform.


1. Read the entire **Before you begin** section.
1. Follow the **Click here** link to enable your subscription. You cannot use Ansible Automation Platform without a valid subscription.

1. Return to the **Overview** tab and clickCreateto initiate the deployment process.
