# 2. Prerequisites for Installing Red Hat Ansible Automation Platform on Microsoft Azure
## 2.7. Entitling your subscription




To access your deployment, you need an existing subscription entitlement that you can assign to your Ansible Automation Platform when you first log in.

For information on obtaining a subscription entitlement and attaching it to Ansible Automation Platform, see [Red Hat Ansible Automation Platform on Microsoft Azure Subscription Entitlement Association](https://access.redhat.com/articles/6761811) .
