# 6. Support for Red Hat Ansible Automation Platform on Microsoft Azure
## 6.4. Policy remediation timeline and enforcement




If Red Hat identifies a policy issue in your environment, the following timeline applies:

-  **Notification period:** Red Hat provides weekly notices to the customer after identifying and reporting a policy issue.
-  **Unsupported state transition:** Environments that are not remediated within two weeks of the initial report move into an unsupported state.
-  **Resource management:** If an environment remains in an unsupported state for 90 consecutive days, Red Hat might scale down the environment infrastructure to minimize resource impact.
