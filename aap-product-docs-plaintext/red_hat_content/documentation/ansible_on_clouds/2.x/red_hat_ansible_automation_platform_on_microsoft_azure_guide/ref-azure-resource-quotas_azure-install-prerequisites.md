# 2. Prerequisites for Installing Red Hat Ansible Automation Platform on Microsoft Azure
## 2.3. Azure resource quotas and infrastructure limits




Microsoft imposes resource limits within each Azure region. The CPU limit is the most likely to impact Red Hat Ansible Automation Platform on Microsoft Azure.

Before you install Ansible Automation Platform on Microsoft Azure, ensure that you have capacity to deploy the managed application into your desired region. Refer to [Ansible Automation Platform on Microsoft Azure infrastructure usage](https://docs.redhat.com/en/documentation/ansible_on_clouds/2.x/html-single/red_hat_ansible_automation_platform_on_microsoft_azure_guide/index#con-azure-infrastructure-usage_azure-intro) for infrastructure requirements.
