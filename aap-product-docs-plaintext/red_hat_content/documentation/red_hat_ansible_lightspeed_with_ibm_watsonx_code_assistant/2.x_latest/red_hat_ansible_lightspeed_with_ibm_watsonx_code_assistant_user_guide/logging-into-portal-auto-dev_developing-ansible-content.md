# 4. Developing Ansible content
## 4.1. Accessing the Ansible Lightspeed portal as an automation developer
### 4.1.1. Logging in to the Ansible Lightspeed portal as an automation developer




**Procedure**

1. Go to the [Ansible Lightspeed portal login page](https://c.ai.ansible.redhat.com/) .
1. ClickLog in→Log in with Red Hat.
1. Enter your Red Hat account username and password.


On successful authentication, the login screen is displayed along with your username and your assigned user role.
