# 8. Troubleshooting
## 8.1. Troubleshooting Red Hat Ansible Lightspeed configuration errors
### 8.1.2. Cannot save the API key




When you enter the IBM watsonx Code Assistant API key, authentication fails and shows the following error message:

`IBM Cloud API key is invalid`

Red Hat Ansible Lightspeed verifies the API key by generating an associated access token. To resolve the error, ensure that you have not accidentally included any extra spaces when obtaining the API key from IBM watsonx Code Assistant. If you still cannot upload the API key, contact [IBM Support](https://cloud.ibm.com/docs/get-support?topic=get-support-open-case) .
