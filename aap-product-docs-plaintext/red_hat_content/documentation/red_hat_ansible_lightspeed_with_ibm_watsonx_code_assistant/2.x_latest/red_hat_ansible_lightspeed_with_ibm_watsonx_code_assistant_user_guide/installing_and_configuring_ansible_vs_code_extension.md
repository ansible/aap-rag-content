# 2. Getting started with Red Hat Ansible Lightspeed
## 2.2. Quick start for developers
### 2.2.1. Installing and configuring Ansible VS Code extension




To access Red Hat Ansible Lightspeed, all Ansible users must install and configure the Ansible VS Code extension in their VS Code editor. For the procedure, see [Install and configure the Ansible VS Code extension](https://docs.redhat.com/en/documentation/red_hat_ansible_lightspeed_with_ibm_watsonx_code_assistant/2.x_latest/html-single/red_hat_ansible_lightspeed_with_ibm_watsonx_code_assistant_user_guide/index#con-configure-vs-code-extension_developing-ansible-content) .
